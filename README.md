# Dotfiles

## Install stow

```shell
brew install stow
```

## Install stow packages

```shell
make # or make <package> to install individual package
```

## ZSH features

```shell
brew install zsh-autosuggestions
brew install zsh-syntax-highlighting
```

## Install oh-my-posh

```shell
brew install jandedobbeleer/oh-my-posh/oh-my-posh
```
